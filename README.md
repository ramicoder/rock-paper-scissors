A simple implementation of the classic game played entirely in the browser console. This project was built as part of **The Odin Project** curriculum (Foundations Course).

## Features
- **5 Rounds:** The game automatically runs for a full set of 5 rounds.
- **Console Interface:** Uses `prompt()` for user input and `console.log()` for results.
- **Score Tracking:** Tracks the human vs. computer score and declares a final winner.
- **Case Insensitive:** Inputs like "rock", "Rock", and "ROCK" are all accepted.

## How to Play
1. Open `index.html` in your web browser.
2. Open the **Developer Tools** (Right-click anywhere > Inspect, or press `F12`).
3. Click on the **Console** tab.
4. Refresh the page to start the game.
5. Type your choice (Rock, Paper, or Scissors) in the prompt window.
